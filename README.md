# Zavrsni-rad
Restaurant Table Reservation System

This is a web application for managing restaurant table reservations. It includes a frontend, backend, and an admin interface.

### How to run the project ###

# 1. Install dependencies

npm install

---

# 2. Start the backend
Navigate to the backend folder and run:


npm run dev


---

# 3. Start the frontend (Rezervacije stolova)
Navigate to the `rezervacije_stolova` folder and run:


npm start


---
